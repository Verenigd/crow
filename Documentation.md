# Documentation
## File format
The file works as such:

    - Entry keys and values are seprated by a `:`
    - Entries are separated by a new lines

```
ISO: AT
Capital: Wien
Geography:
    LargestPeak: Alps
    Climate:
        Primary: Dfb
        Secondary: Dfc
        Tertiary: ET
    :
    Continent: Europe
    Landlocked: true
:
```
